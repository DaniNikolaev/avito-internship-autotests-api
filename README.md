# Автоматизированные тесты API Avito Internship

Этот проект реализует автоматизированные тесты для API от команды стажировок Avito. Тесты написаны с использованием **Python**, **Pytest**, **Allure**, **Pydantic**, **Faker** и **HTTPX**. Проект покрывает позитивные и негативные сценарии для эндпоинтов объявлений и статистики версий v1 и v2.

## Обзор проекта

Цель этого проекта — автоматизировать тестирование API от Avito Internship. Автоматизированные тесты проверяют различные функции приложения, чтобы обеспечить его стабильность и корректность.

Проект разработан для автоматических тестов REST API. Используются следующие технологии:

-   **API клиенты** для структурированного взаимодействия с конечными точками (разделение по версиям и доменам),
-   **Фикстуры Pytest** для переиспользуемых и поддерживаемых тестовых настроек,
-   **Модели Pydantic** для строгой валидации данных и генерации JSON-схем,
-   **Валидация схем** для обеспечения соответствия контракту API,
-   **Генерация фейковых данных** (Faker) для имитации реальных сценариев,
-   **Allure** для подробных и наглядных отчетов,
-   **Идемпотентность** — тесты для проверки идемпотентности GET и DELETE запросов,
-   **Негативные сценарии** — null, пустые значения, некорректные типы данных.
-   **Линтеры кода** - для проверки качества кода и соответствия стилю PEP 8

## Начало работы

### Клонирование репозитория

Чтобы начать, клонируйте репозиторий проекта с помощью Git:

```bash
git clone https://github.com/DaniNikolaev/avito-internship-autotests-api.git
cd avito-internship-autotests-api
```

### Создание виртуального окружения

Рекомендуется использовать виртуальное окружение для управления зависимостями проекта. Следуйте инструкциям для вашей
операционной системы:

#### Linux / MacOS

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Установка зависимостей

После активации виртуального окружения установите зависимости проекта, перечисленные в файле `requirements.txt`:

```bash
pip install -r requirements.txt
```

### Запуск тестов с генерацией Allure-отчета

Чтобы запустить тесты и сгенерировать Allure-отчет, используйте следующую команду:

```bash
pytest -m "regression" --alluredir=./allure-results
```

Это выполнит все тесты в проекте и отобразит результаты в терминале.

### Запуск линтеров кода

Чтобы запустить линтеры кода, используйте следующие команды:

```bash
flake8 . --config=.flake8
pylint . --rcfile=.pylintrc
```

Это выполнит анализ кода и отобразит результаты в терминале.

### Просмотр Allure-отчета

После выполнения тестов можно сгенерировать и просмотреть Allure-отчет с помощью:

```bash
allure serve allure-results
```

Эта команда откроет Allure-отчет в вашем браузере по умолчанию.
